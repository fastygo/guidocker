Notes

  • These settings apply only to managed application domains, not to the dashboard itself.
  • TLS settings are applied only when "Enable HTTPS on proxy" is enabled for an app.
  • Certificates are issued and removed against the host installation of certbot.
  • Host nginx owns ports 80/443 and keeps routing active even if the dashboard container is removed.
