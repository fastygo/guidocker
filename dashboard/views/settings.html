Settings
Manage platform endpoint settings used by the admin API and external access.
Admin panel endpoint

Platform stores this preference, but certificate renewal is expected to run on the host (for example with cron using certbot). Example: certbot renew --quiet --post-hook "nginx -s reload"

Notes
  • Platform exposes this config through /api/settings.
  • When domain is not set, the panel is reachable via host IP.
  • TLS settings are applied when the stack config is attached to a proxy.
  • Renewals should be driven by host-level automation (systemd timer / cron) and then reload nginx.
